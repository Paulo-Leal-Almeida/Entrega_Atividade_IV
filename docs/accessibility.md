# Acessibilidade (WCAG 2.1) - Resumo das Implementações

- Navegação por teclado (tabindex, handlers para Enter/Space)
- Skip link para pular para o conteúdo
- Foco visível (outline forte)
- Uso de roles e aria-labels em landmarks e controles
- Mensagens de erro com role="status" / aria-live="polite"
- Modos: dark-mode e high-contrast
- Uso de loading="lazy" para imagens
